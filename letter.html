Рабочий процесс

1. Ресерч и план

Глубокий ресерч темы Opus'ом, декомпозиция задачи и подготовка плана перед исполнением.

2. Cursor-агенты

План уходит агентам на реализацию. Для каждого проекта есть свои правила: стиль, архитектура и код-конвенции.

3. Хуки, тесты и ревью

Pre/post-хуки запускают форматтер и тесты, а перед коммитом использую cross-family review моделью другого семейства.

4. TDD / SDD и интеграция

Часто начинаю со спек и тестов, потом пишется функциональный код, полируется логика и собирается готовое решение.

Личные агенты через NeuroGate API

В CherryStudio через NeuroGate API крутятся мои личные агенты: SEO-агент по NeuroGate, аналитический агент, и агент-смотритель за релизами новых моделей, который сам добавляет их в каталог сервиса.
